### Description
![SignalVariableTimeDelay picture](SignalVariableTimeDelay.svg)

Delays input variable with specified time delay

#### Input Variables
* **maxMem** - Maximum allowed memory consumption [MB]
* **dT** - Time delay [s]
* **in** - Input variable [-]

#### Output Variables
* **out** - Delayed variable [-]

### Theory
<!---EQUATION out(t) = in(t - dT)--->

### Description
![SignalSegare picture](SignalSegareHelp.svg =150x)

Computes a signal circular segment area


#### Input Variables
* **x** - Segment height [-]
* **diameter** - Segment diameter [-]

#### Output Variables
* **area** - Segment area [-]
